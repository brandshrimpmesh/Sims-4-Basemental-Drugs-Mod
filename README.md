## 📥 Installation

### 1. Get the Latest Release

Download the latest Basemental Drugs package from the official releases page

### 2. Install the Mod

* Extract the downloaded archive.
* Open your Sims 4 Mods folder:

  ```
  Documents/Electronic Arts/The Sims 4/Mods
  ```
* Copy the contents of the Basemental Drugs package into the Mods folder.
* Follow any installation notes included with the download.

### 3. Enable Mods In-Game

* Launch The Sims 4.
* Open **Game Options → Other**.
* Enable:

  * **Custom Content and Mods**
  * **Script Mods Allowed**
* Restart the game after enabling these settings.

### 4. Verify Installation

* Load a household.
* Click on a Sim, computer, or phone to check for Basemental Drugs menu options.
* If features are not available, review the troubleshooting section below.

## ⚠️ Troubleshooting & Common Issues

### ✔ Basemental Drugs Not Showing

* Verify **Script Mods Allowed** and **Custom Content and Mods** are enabled.
* Ensure mod files are placed directly in the Mods folder and not excessively nested.
* Delete:

  ```
  localthumbcache.package
  ```

  from your Sims 4 directory.
* Remove duplicate copies of Basemental Drugs.

### ✔ Script Errors or Missing Features

* Download and install the latest version of Basemental Drugs.
* Remove outdated or conflicting mods.
* Check compatibility with your current Sims 4 game version.
* Restart the game after making changes.

### ✔ Version Mismatch / Compatibility Problems

* After major Sims 4 updates, install the newest Basemental Drugs version.
* Avoid mixing files from different Basemental releases.
* Review update notes for known compatibility issues.
